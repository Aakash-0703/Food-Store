# 🍔 Food Munch Website

A responsive and modern **Food Ordering Website UI** built using **HTML** and **Tailwind CSS**. This project showcases a clean layout, responsive design, and attractive UI components for a food service platform.

---
## 📌 Features

* ✅ Fully Responsive Design (Mobile, Tablet, Desktop)
* ✅ Modern UI using Tailwind CSS
* ✅ Hero Section with Background Image
* ✅ "Why Choose Us" Section
* ✅ Explore Menu Grid Layout
* ✅ Food Categories Cards
* ✅ Delivery & Payment Section
* ✅ Customer Appreciation Section
* ✅ Clean Footer Design

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS (CDN)**

## 💡 How to Use

1. Clone the repository

```bash
git clone https://github.com/your-username/food-munch.git
```

2. Open the project folder

```bash
cd food-munch
```

3. Run the project

* Simply open `index.html` in your browser
